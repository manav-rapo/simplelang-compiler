# 8bit-cpu-compiler
8bit-cpu/compiler
# run compiler

move to this path:

```bash
cd compiler
```
run the compiler:

```bash
g++ lexer.cpp parser.cpp codegenerator.cpp compiler.cpp -o compiler
```
after running g++ command now run main file :

```bash
./compiler
```
you can give any input on input.txt file after running g++ command
